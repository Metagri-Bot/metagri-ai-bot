# Metagri研究所 軌跡・歴史 公開用サマリー

公開可否: public
更新日: 2026-04-25

## 一言で

Metagri研究所は、2022年3月に当初5名でスタートし、2026年時点で1,300名を超えるDiscordコミュニティに育った、農業×新技術の実験コミュニティです。

## 公開してよい説明

- 2022年3月: 株式会社農情人が運営するMetagri研究所がスタート。当初は5名規模。
- 2022年4月以降: 農業NFTプロジェクトを順次立ち上げ、スイカ、トマト、シャインマスカット、いちご、マンゴー、柑橘の接ぎ木などのNFTを発行。
- MLTT（MetagriLabo Thanks Token）を発行し、コミュニティへの貢献を可視化する仕組みを開始。
- 毎週日曜の定例MTGや、テーマ別の交流会、AIハッカソンなどを継続的に実施。
- 2026年3月: 4年間DAOレポートを無料公開し、5名から1,300名規模へ成長したコミュニティ実績を整理。
- 2026年: 自治体連携として、白井市PR動画コンテストを実施。動画生成AIの活用事例を地域に広げる試み。
- 2026年: 農業AIハッカソン、未来の農業シミュレーター（Roblox）、農業AI通信などを並行展開。

## 主なターニングポイント

- 農業NFTの発行開始（2022年4月）: 農業×web3の実装スタート。
- MLTT / MLTG導入: 貢献を可視化するトークンエコノミーの形成。
- 1,000名規模到達: 単発のオンライン企画ではなく、継続的コミュニティとして定着。
- 4年間DAOレポート公開（2026年3月）: 「続くDAO」の根拠データを公開。
- 自治体連携の本格化: 白井市PR動画コンテストなど、地域連携の事例化。

## 過去の主な活動カテゴリ

- 毎週日曜の定例MTG（コミュニティ運営の中心軸）
- 農業NFT発行（複数の農産物テーマ）
- AIハッカソン / Vibe Codingワークショップ
- 動画生成AIコンテスト
- 出版・メディア発信（Kindle、商業出版、ラジオ出演など）
