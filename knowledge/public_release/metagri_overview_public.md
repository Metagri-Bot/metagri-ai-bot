# Metagri研究所 公開用サマリー

公開可否: public
更新日: 2026-04-25

## 一言で

Metagri研究所は、株式会社農情人が運営する、農業×新技術の実験コミュニティです。Discordを中心に、web3、生成AI、メタバース、自治体連携、教育活動などを組み合わせ、新しい農業の関わり方を実証しています。

## 公開してよい説明

- コンセプトは「Meta（超越）× Agriculture（農業）」。
- 農業の固定観念をテクノロジーで超越することを掲げている。
- 主な取り組みは、農業AI通信、白井市PR動画コンテスト、MLTT / MLTG、未来の農業シミュレーター、農業AIハッカソン、4年間DAOレポート。

## 外部公開時の注意

- Discord内部の未公開議事録、Notionの検討資料、担当者名、売上・契約・個別相談内容は出さない。
- 進行中企画は、公開ページがあるものだけ具体名を出す。
